在github上搭建hexo博客

发表于 2017-08-26 |

一、 前言

这是一篇使用GitHub Pages和Hexo搭建免费独立博客的总结。

作为一个小白，按照网上的各种教程搭建hexo遇到一堆问题，对版本控制一点概念都没有，更别说使用GitHub Page能做出一个好看又好用的博客了。所以第一次的尝试失败了。

这个周末总算将他给弄出来了，这次倒是顺风顺水，可能跟我刚重装了系统有关吧哈哈

我所总结的是本地搭建hexo并推到GitHub Pages上并绑定独立域名的过程。

如果是小小白，可以先花时间去了解下：

  • Git
  • GitHub
  • GitHub Pages
  • Hexo

在windows下安装git比较常用的有两种方式：

  1. Git 官方版本的安装
  2. GitHub for Windows

2.2.2 配置 Git

当安装完Git应该做的第一件事情就是设置用户名称和邮件地址。这样做很重要，因为每一个Git的提交都会使用这些信息，并且它会写入你的每一次提交中，不可更改：
